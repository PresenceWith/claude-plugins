# 데이터 분석 이해도 검증 퀴즈

## 개요

데이터 분석에 대해 학습한 내용을 검증하기 위한 퀴즈입니다. 총 5문제로 구성되어 있으며, 각 문제에 대한 답변과 피드백을 포함합니다.

---

## 문제 1: 기술 통계량

**다음 중 데이터의 중심 경향을 나타내는 대표적인 통계량 3가지를 고르세요.**

A) 평균(Mean), 중앙값(Median), 최빈값(Mode)  
B) 분산(Variance), 표준편차(Standard Deviation), 범위(Range)  
C) 왜도(Skewness), 첨도(Kurtosis), 공분산(Covariance)  
D) 사분위수(Quartile), 백분위수(Percentile), IQR

**사용자 답변:** A) 평균, 중앙값, 최빈값

**정답:** A

**결과: 정답**

**피드백:** 정확합니다. 평균, 중앙값, 최빈값은 데이터의 중심 경향(Central Tendency)을 측정하는 대표적인 통계량입니다. B는 산포도(Dispersion) 측정 지표들이고, D는 위치 통계량(Positional Statistics)입니다. 데이터의 분포 형태에 따라 어떤 중심 경향 측정치가 더 적합한지 판단하는 것이 중요합니다. 예를 들어, 극단값이 있는 경우 중앙값이 평균보다 더 안정적인 지표가 됩니다.

## 문제 2: 상관관계와 인과관계

**"아이스크림 판매량과 익사 사고 건수 사이에 양의 상관관계가 있다"는 분석 결과에 대해 올바른 해석은?**

A) 아이스크림을 많이 먹으면 익사 사고가 증가한다  
B) 익사 사고가 많아지면 아이스크림 판매가 증가한다  
C) 두 변수 모두 제3의 변수(예: 기온)에 의해 영향을 받는 교란 변수(Confounding Variable)의 영향일 수 있으므로, 상관관계만으로 인과관계를 단정할 수 없다  
D) 데이터 수집 오류이므로 무시해야 한다

**사용자 답변:** C) 교란 변수의 영향일 수 있으므로 인과관계를 단정할 수 없다

**정답:** C

**결과: 정답**

**피드백:** 정확합니다. 이것은 데이터 분석에서 가장 중요한 원칙 중 하나인 "상관관계는 인과관계를 의미하지 않는다(Correlation does not imply causation)"의 대표적인 예시입니다. 여기서 기온이 교란 변수(Confounding Variable)로 작용하여 두 변수 모두에 영향을 미칩니다. 인과관계를 확립하려면 무작위 통제 실험(RCT) 등의 실험 설계가 필요합니다.

## 문제 3: 데이터 전처리

**결측치(Missing Values)를 처리하는 방법으로 적절하지 않은 것은?**

A) 결측치가 포함된 행을 삭제한다 (Listwise Deletion)  
B) 평균값이나 중앙값으로 대체한다 (Imputation)  
C) 결측치가 있는 열 전체를 항상 삭제한다  
D) 다중 대체법(Multiple Imputation)을 사용한다

**사용자 답변:** B) 평균값이나 중앙값으로 대체한다

**정답:** C

**결과: 오답**

**피드백:** 아쉽습니다. B는 결측치 처리의 유효한 방법 중 하나입니다. 단순 대체법(평균/중앙값 대체)은 데이터의 분산을 과소추정할 수 있다는 한계가 있지만, 결측 비율이 낮고 무작위 결측(MCAR)인 경우에는 실무에서 자주 사용됩니다.

정답은 C입니다. 결측치가 있는 열 전체를 "항상" 삭제하는 것은 적절하지 않습니다. 결측 비율이 극히 높거나 분석에 중요하지 않은 변수일 경우에만 열 삭제가 정당화됩니다. 결측치 처리는 결측 패턴(MCAR, MAR, MNAR)을 먼저 파악한 후, 상황에 맞는 방법을 선택해야 합니다.

**핵심 학습 포인트:** 결측치 처리에는 "항상 옳은" 하나의 방법이 없습니다. 결측 비율, 결측 패턴, 변수의 중요도를 종합적으로 고려하여 방법을 선택해야 합니다.

## 문제 4: 시각화

**히스토그램(Histogram)과 막대 그래프(Bar Chart)의 차이점에 대한 설명으로 올바른 것은?**

A) 히스토그램은 연속형 데이터의 분포를 나타내며, 막대 그래프는 범주형 데이터의 빈도를 나타낸다  
B) 히스토그램과 막대 그래프는 동일한 차트이며 이름만 다르다  
C) 막대 그래프는 연속형 데이터에, 히스토그램은 범주형 데이터에 사용한다  
D) 히스토그램은 항상 정규분포 형태를 보인다

**사용자 답변:** A) 히스토그램은 연속형, 막대 그래프는 범주형

**정답:** A

**결과: 정답**

**피드백:** 정확합니다. 히스토그램은 연속형 데이터를 구간(bin)으로 나누어 각 구간에 속하는 데이터의 빈도를 표시합니다. 막대 사이에 간격이 없으며, 데이터의 분포 형태를 파악하는 데 사용합니다. 반면 막대 그래프는 범주형 데이터의 각 카테고리별 값을 비교하며, 막대 사이에 간격이 있습니다. 시각화 도구를 선택할 때 데이터의 유형을 먼저 파악하는 것이 중요합니다.

## 문제 5: 가설 검정

**p-value가 0.03이고 유의수준(alpha)이 0.05일 때, 올바른 해석은?**

A) 귀무가설이 참일 확률이 3%이다  
B) 유의수준보다 p-value가 작으므로 귀무가설을 기각한다  
C) 대립가설이 참일 확률이 97%이다  
D) 표본 크기가 충분하지 않다

**사용자 답변:** A) 귀무가설이 참일 확률이 3%이다

**정답:** B

**결과: 오답**

**피드백:** 아쉽습니다. 이것은 데이터 분석에서 가장 흔한 오해 중 하나입니다.

p-value는 "귀무가설이 참일 확률"이 아닙니다. p-value의 정확한 정의는 **"귀무가설이 참이라고 가정했을 때, 관찰된 데이터 또는 그보다 더 극단적인 결과를 얻을 확률"**입니다.

p-value(0.03) < 유의수준(0.05)이므로, 귀무가설을 기각하고 통계적으로 유의한 결과라고 판단합니다. 그러나 이것이 대립가설이 참이라는 것을 "증명"하는 것은 아닙니다.

**핵심 학습 포인트:** p-value의 올바른 해석은 매우 중요합니다. "귀무가설이 참일 확률"이라는 해석은 대표적인 통계적 오류입니다. 또한 통계적 유의성(statistical significance)과 실질적 유의성(practical significance)은 다른 개념이므로, 효과 크기(effect size)도 함께 고려해야 합니다.
