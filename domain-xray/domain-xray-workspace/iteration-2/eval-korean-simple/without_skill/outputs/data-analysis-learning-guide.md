# 데이터 분석 분야 학습 가이드

## 데이터 분석이란?

데이터 분석은 원시 데이터에서 유의미한 패턴, 인사이트, 결론을 도출하는 과정입니다. 비즈니스 의사결정, 과학 연구, 정책 수립 등 거의 모든 분야에서 활용됩니다.

---

## 1단계: 기초 역량 다지기

### 통계학 기초
- **기술통계**: 평균, 중앙값, 표준편차, 분산
- **확률론**: 확률분포(정규분포, 이항분포 등), 조건부 확률, 베이즈 정리
- **추론통계**: 가설검정, 신뢰구간, p-value
- **상관분석과 회귀분석**: 변수 간 관계 파악

추천 자료:
- "통계학 입문" (모든 대학 통계 교재가 적합)
- Khan Academy 통계 강좌 (무료, 영어지만 한글 자막 지원)

### 수학 기초
- 선형대수: 행렬 연산, 벡터 공간 (머신러닝 기초)
- 미적분: 최적화 이해에 필요
- 기초 수준이면 충분하며, 필요할 때 심화하면 됩니다

## 2단계: 도구 학습

### Python (가장 추천)
데이터 분석의 사실상 표준 언어입니다.

핵심 라이브러리:
- **pandas**: 데이터 조작 및 정제 (테이블 형태 데이터 처리)
- **NumPy**: 수치 연산
- **Matplotlib / Seaborn**: 데이터 시각화
- **scikit-learn**: 머신러닝 (분류, 회귀, 클러스터링)

```python
# 예시: pandas로 데이터 불러오기
import pandas as pd

df = pd.read_csv("sales_data.csv")
print(df.describe())  # 기술통계 한눈에 보기
print(df.groupby("region")["revenue"].mean())  # 지역별 평균 매출
```

### SQL
데이터베이스에서 데이터를 추출하는 필수 기술입니다.

```sql
-- 월별 매출 집계 예시
SELECT 
    DATE_TRUNC('month', order_date) AS month,
    SUM(amount) AS total_revenue,
    COUNT(DISTINCT customer_id) AS unique_customers
FROM orders
WHERE order_date >= '2025-01-01'
GROUP BY 1
ORDER BY 1;
```

## 3단계: 분석 프로세스 익히기

실무에서 데이터 분석은 다음 순서로 진행됩니다:

### 1. 문제 정의
- 무엇을 알고 싶은가? 어떤 의사결정에 도움이 되는가?
- 예: "고객 이탈률이 왜 증가했는가?"

### 2. 데이터 수집
- 데이터베이스, API, 웹 스크래핑, 설문조사 등
- 어떤 데이터가 필요하고, 어디서 구할 수 있는지 파악

### 3. 데이터 정제 (전체 시간의 60~80%)
- 결측값 처리 (삭제, 대체, 보간)
- 이상치 탐지 및 처리
- 데이터 타입 변환, 중복 제거
- 일관성 검증

### 4. 탐색적 데이터 분석 (EDA)
- 데이터의 분포, 패턴, 이상 징후를 시각화로 확인
- 변수 간 상관관계 탐색
- 가설 수립

### 5. 분석 및 모델링
- 통계 검정, 회귀 분석, 머신러닝 모델 구축
- 목적에 맞는 적절한 기법 선택

### 6. 결과 해석 및 커뮤니케이션
- 분석 결과를 비전문가도 이해할 수 있게 전달
- 시각화와 스토리텔링 활용
- 한계점과 가정 명시

## 4단계: 실전 프로젝트로 경험 쌓기

### 입문 프로젝트 아이디어
1. **공공데이터 분석**: 공공데이터포털(data.go.kr)에서 데이터를 받아 탐색적 분석 수행
2. **개인 데이터 분석**: 본인의 소비 내역, 운동 기록 등을 분석
3. **Kaggle 경진대회**: 타이타닉 생존자 예측, 주택 가격 예측 등 입문 대회 참가

### 중급 프로젝트
4. **A/B 테스트 분석**: 두 그룹 간 차이의 통계적 유의성 검정
5. **시계열 분석**: 주가, 기온, 판매량 등의 추세와 계절성 분석
6. **고객 세분화**: 클러스터링을 활용한 고객 그룹 분류
